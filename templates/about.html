Project Overview

About Steam Success Predictor

This project predicts whether a Steam game is likely to be successful by using machine learning and game metadata.

← Predictor Feature Importance

🎯 Project Goal

The goal of this project is to estimate the success potential of Steam games using store page information, playtime statistics, game tags and platform data.

🧠 Machine Learning Model

A Random Forest classification model is used to predict whether a game belongs to the successful or unsuccessful class.

📊 Dataset

The project uses Steam game data and SteamSpy tag features. The dataset includes game metadata, ratings, playtime values, platform support and tag-based features.

⚙️ Technologies

Python, Flask, PostgreSQL, SQLAlchemy, Pandas, Scikit-learn, HTML, CSS and JavaScript were used during the development process.

🔍 Explainability

The Feature Importance page shows which variables have the strongest impact on the model's predictions.

🗄 Prediction History

Each prediction is saved in a PostgreSQL database. Users can view previous predictions, success probabilities and selected tags.
